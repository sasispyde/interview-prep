CSS Cheet Sheet

CSS Selector
S.No Selector Description Example
1 ID The element on the page with the specified ID. On a given HTML page, each id value should be unique.( use # ) #demo ( <p id="demo" > </p> )
2 CLASS The element(s) on the page with the specified class. Multiple instances of the same class can appear on a page.(use .) .demo ( <p class="demo" > </p> )
3 ELEMENT All HTML elements of the specified type.( use tag name ) p ( <p class="demo" > </p> )
4 ATTRIBUTE Selects the element with the specified attribute p[name="demo"] ( <p name="demo" > </p> )
5 PSEUDO The specified element(s), but only when in the specified state. (For example, when a cursor hovers over a link.) p:hover ( <p name="demo" > </p> )
6 UNIVERSAL It selects everything in the document (or inside the parent element if it is being chained together with another element and a descendant combinator). *
